HTML Anchor Element

The HTML anchor element (<a>) defines a hyperlink, the target of a link, or both.

Attributes Associated with Anchor Element

Example on 'href' Attribute

The href attribute specifies the URL of the page the link goes to.

GOOGLE

Example on 'target' Attribute

The target attribute specifies where to open the linked document.
The target attribute can have one of the following values:

  • _blank - Opens the linked document in a new window or tab (this is default).
    GOOGLE

  • _self - Opens the linked document in the same window/tab as it was clicked.
    GOOGLE
